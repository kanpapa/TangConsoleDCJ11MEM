# RT-11 version4

- 試行錯誤の末、若干のworkaroundを追加することによって起動させることができました。
- ちゃんと使うにはフロッピーディスクのエミュレーションが必要そうなのですが、プロトコルが面倒そうなのでまだ作っていません。

- top.vでコメントアウトされている下記の行を有効にしてビルドします。
```
`define USE_RT11V4_WORKAROUND  // uncomment this line to boot RT-11 v4
```
## sdメモリイメージ
- wfjm's Home[System Images](https://wfjm.github.io/home/w11/inst/systems.html)にある RT11 V4.0 rkset にある rtv4_rk.dsk から、下記の手順でsdメモリ用イメージを作ます。
```
dd if=/dev/zero of=sd-rtv4.dsk count=1024
dd if=rtv4_rk.dsk of=sd-rtv4.dsk seek=1024 conv=notrunc
```
- これをddでsdメモリの先頭に書き込みます。デバイス名はちゃんと調べて絶対に間違えないようにして下さい。間違えるとパソコンの記憶領域を破壊する可能性があります。
```
dd if=sd-rtv4.dsk of=/dev/xxx  ←xxxはsdメモリのデバイス名に直して下さい。
```
